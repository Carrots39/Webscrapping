Installing Guide
1. setup NodeJS
in console / terminal
goto backend : npm install -> npm start
goto frontend : npm install -> npm start

Using Guide 

first : select scrapping Unit
that means how many days can be scrapped at once.
it is depends on your computer.

second : select date range and click "Scrape Matches"

third : if scrapping failed in some days, select that day or range and click "Scrape More"

fourth : all scrapped : Click Scrape BoxScore

fifth: click Export CSV
